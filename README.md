### Telegram bot for computer control
Bot to control sound, reboot and shutdown
### ENV  
In the environment variables, you need to set the bot API token  
`BOT_TOKEN`="Telegram token"
### SLASH COMMANDS  
`/start` - Command for displaying control buttons
### RUN  
 - Need Python 3.8
 - python run main.py
 - Telegram
### USE  
 -  Run main.py
 -  Click on the button you need
